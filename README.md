# Разработка алгоритма реранкинга мультимодальных данных 

# Цель проекта
Разработать и оценить алгоритм реранкинга мультимодальных данных (текст + изображение/PDF страницы) для улучшения поиска в RAG-системах.

# Техническое задание
- **Входные данные:** Текст + изображения/PDF страницы
- **Выход:** Улучшенный порядок кандидатов после первичного поиска
- **Метрики:** Recall@k, nDCG@k, MRR, точность атрибуции, латентность (p50/p95)
- **Результат:** Модуль ре-ранкинга с API, скрипты обучения/инференса, набор экспериментов и отчёт с сравнениями против текстового ре-ранкера
